## 📜 Guiding Principles

1. **Understand, don't memorise** — trace every algorithm by hand before coding it
2. **Big-O first** — always state the time/space complexity of every solution
3. **Two solutions minimum** — brute force first, then optimise
4. **Daily > binge** — 1.5 hrs/day beats 10 hrs on Sunday
